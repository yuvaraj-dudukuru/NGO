## How to view any page
Double-click the `.html` file to open it in a browser, or in VS Code right-click →
**"Open with Live Server"**. Markdown files (`.md`) render best on GitHub or in a
Markdown preview.

## What Day 2 covered
HTML5 document structure · headings & paragraphs · text-formatting tags · images &
`alt` text · links (external/email/anchor) and `target="_blank"` safety · lists ·
tables with `colspan`/`rowspan` · forms & labels · semantic, accessible, valid markup.
