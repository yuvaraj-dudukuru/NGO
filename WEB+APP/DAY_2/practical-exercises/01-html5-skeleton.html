HTML5 Skeleton

This page is the basic HTML5 skeleton recreated from memory. The essential parts are:

  • <!DOCTYPE html> — declares the document as HTML5.
  • <html lang="en"> — the root element, with the page language.
  • <head> — metadata: character set, viewport, and title.
  • <meta charset="UTF-8"> — the character encoding.
  • <meta name="viewport" ...> — responsive scaling on mobile.
  • <title> — the text shown on the browser tab.
  • <body> — all the visible page content.
