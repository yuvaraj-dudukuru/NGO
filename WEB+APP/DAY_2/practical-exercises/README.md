# Practical Exercises — HTML Basics

Five standalone HTML pages. Open any file in a browser to view it.

| File | Exercise |
|------|----------|
| `01-html5-skeleton.html` | HTML5 skeleton recreated from memory (compare against Section 4.3) |
| `02-headings.html` | All six heading levels, each with a paragraph explaining its purpose |
| `03-text-formatting.html` | Every text-formatting tag (`strong`, `b`, `em`, `i`, `mark`, `small`, `sub`, `sup`) with a label |
| `04-lists.html` | Ordered, unordered, and description lists (4+ items each) plus one nested list |
| `05-timetable.html` | Weekly timetable using `<thead>`, `<tbody>`, and both `colspan` and `rowspan` |

## Self-check for Exercise 1
After recreating the skeleton from memory, compare it to Section 4.3. The
required pieces are: `<!DOCTYPE html>`, `<html lang>`, `<head>` with
`<meta charset>`, `<meta name="viewport">`, `<title>`, and `<body>`.
