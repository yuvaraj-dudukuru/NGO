# Personal Portfolio — Mini Project Demo

A single-page personal portfolio built with semantic HTML5 and CSS.

## Files
- `index.html` — the portfolio page
- `style.css` — styling

## How to view
Open `index.html` in any web browser (double-click it).

## Tasks covered
1. **Portfolio page** — name, photo (placeholder), about section, education
   table, skills list, and a contact form.
2. **Navigation menu** — the nav links scroll to each section using `id`
   anchors (`#home`, `#about`, `#education`, `#skills`, `#media`, `#contact`).
3. **Link types**
   - External link → opens safely in a new tab
     (`target="_blank" rel="noopener noreferrer"`).
   - Email link → `mailto:alex.carter@example.com`.
   - Telephone link → `tel:+910000000000`.
4. **Embedded media** — both a `<video controls>` and an `<audio controls>`
   element are included.

## Task 5 — Validate with the W3C Markup Validation Service
1. Go to **https://validator.w3.org/**.
2. Choose the **"Validate by File Upload"** tab and upload `index.html`
   (or use "Validate by Direct Input" and paste the HTML).
3. Click **Check** and fix any errors it reports.

> Tip: replace the placeholder photo, university names, and phone number with
> your real details before submitting.
