# Mini-Project 2 — Product Landing Page

A product landing page (the "FocusFlow" demo): a hero, a features grid, a
call-to-action, and a footer. All copy and branding are placeholder/demo content.

## Files
- `index.html` — hero section, a 6-card features grid, a CTA band, and a footer.
- `styles.css` — responsive styling.

## How it works
The **hero** introduces the product; the **features** sit in a responsive grid
that drops from three columns to one on small screens; a **call-to-action** band
and **footer** close the page.

## How to run
Open `index.html` in a browser (double-click or **Live Server**), then resize /
use device mode to see the features grid reflow.

## How to stop
Close the browser tab.

## Concept
A marketing-style page layout: hero → features grid → CTA → footer.
