# Mini-Project 1 — Responsive Portfolio

The personal portfolio from earlier days, reworked to be **fully responsive** —
it looks good on phones, tablets, and desktops. All content is placeholder/demo data.

## Files
- `index.html` — navbar, hero, About, Skills, Projects, and Contact sections.
- `styles.css` — responsive styling with Flexbox/Grid and media queries.

## How it works
Sections use Flexbox/Grid that reflow as the viewport shrinks; media queries
adjust spacing, font sizes, and column counts at tablet and phone breakpoints.

## How to run
Open `index.html` in a browser (double-click or **Live Server**), then resize the
window or use device mode (**Ctrl+Shift+M**).

## How to stop
Close the browser tab.

## Note
Email/contact details are placeholders — replace them with your own.
