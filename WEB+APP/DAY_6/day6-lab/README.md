# Day 6 Lab — Responsive Design Step by Step

The hands-on Day 6 lab. It builds a responsive page in numbered steps, covering
the viewport tag, a Flexbox navbar, a centered hero, a responsive card grid, and
media queries.

## Files
- `index.html` — the page: navbar, hero, and a grid of cards.
- `styles.css` — the styling, organised by lab step.

## What it does (steps)
1. **Viewport & base** — the `<meta name="viewport">` tag makes the page scale on phones.
2. **Flexbox navbar** — a horizontal bar that spaces the logo and links.
3. **Centered hero** — a full-width banner with centered content.
4. **Responsive card grid** — cards that reflow from multiple columns to one.
5. **Media queries** — `@media (max-width: 600px)` tweaks for small screens.

## How to run
1. Open `index.html` in a browser (double-click or **Live Server**).
2. Press **F12**, then **Ctrl+Shift+M** to toggle device mode.
3. Try widths of 375px, 768px, and 1440px to watch the layout reflow.

## How to stop
Close the browser tab (or stop Live Server from the VS Code status bar).

## Concepts practised
Viewport meta tag, Flexbox, centering, responsive grids, and media queries.
