# `lib/` — ui_challenges_demo Source

Holds the Dart source for the five Day 10 UI challenges.

## Files
| File | What it is |
|------|------------|
| `main.dart` | A menu (`MenuScreen`) that opens each challenge: `ProfileCardChallenge` (Card + Row + Expanded), `LoginFormChallenge` (a `StatefulWidget` with two `TextField`s and a password toggle), `SettingsChallenge` (a `ListView` of `ListTile`s), `ProductGridChallenge` (`GridView.builder`), and `FullScaffoldChallenge` (AppBar actions + Drawer + FAB). |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
