# `lib/` — student_management_app Source

Holds the Dart source for the Section 31 demo app.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app. `StudentManagementApp` sets a theme and named routes. Screens: `HomeScreen` (hub), `StudentListScreen`, `StudentDetailsScreen` (receives a student via its constructor), `ProfileScreen`, and `SettingsScreen`. |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
