# `lib/` — Portfolio App Source

Holds the Dart source for Mini Project 4.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app. `PortfolioApp` registers named routes for `PortfolioHomeScreen`, `ProjectsScreen`, and `ContactScreen`. The home screen navigates to the others with `Navigator.pushNamed`. |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
