# `lib/` — Recipe App Source

Holds the Dart source for Mini Project 2.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app. A top-level `recipes` list holds the sample data. `RecipeListScreen` shows the list; tapping a row pushes `RecipeDetailsScreen`, which renders the header, chips, ingredients, and numbered steps. |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
