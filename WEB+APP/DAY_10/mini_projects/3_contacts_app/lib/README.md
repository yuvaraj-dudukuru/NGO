# `lib/` — Contacts App Source

Holds the Dart source for Mini Project 3.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app. A top-level `contacts` list holds the sample data. `ContactListScreen` shows the divided list; tapping a row pushes `ContactDetailsScreen`, which shows the avatar, name, and phone/email/city rows. |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
