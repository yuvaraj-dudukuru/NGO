# `lib/` — Student Task Manager Source

Holds the Dart source for Mini Project 5.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app. Defines a `Task` model and four screens: `TaskHomeScreen`, `TaskListScreen` (a `StatefulWidget` holding the task list), `TaskDetailsScreen`, and `AddTaskScreen` (a `StatefulWidget` form that returns a new `Task` on pop). |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
