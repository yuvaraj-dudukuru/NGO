# `lib/` — Student Management App Source

Holds the Dart source for Mini Project 1.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app. `main()` runs `StudentManagementApp` (a `MaterialApp` with a theme and named routes). Screens: `HomeScreen`, `StudentListScreen`, `StudentDetailsScreen` (receives a student via its constructor), `ProfileScreen`, and `SettingsScreen`. |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
