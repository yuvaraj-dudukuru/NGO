# `lib/` — student_app Source

Holds the Dart source for the Day 10 practical-lab app.

## Files
| File | What it is |
|------|------------|
| `main.dart` | The entire app: `MyApp` (root), `HomeScreen` (a button that pushes the list), `ListScreen` (a `ListView.builder` of names), and `DetailsScreen` (receives the tapped name via its constructor). |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for the full walkthrough.
