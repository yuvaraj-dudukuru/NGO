# `lib/` — navigation_challenges_demo Source

Holds the Dart source for the five Day 10 navigation challenges.

## Files
| File | What it is |
|------|------------|
| `main.dart` | `NavigationChallengesApp` registers named routes and shows a `MenuScreen`. The five challenges: **(1)** push/pop between two screens, **(2)** the same flow via named routes, **(3)** a list that passes data to a details screen, **(4)** a hub that opens three colored screens (named routes), and **(5)** a picker that returns a value with `Navigator.pop(context, value)`. |

## How to run / stop
From the project root (one level up): `flutter pub get` then `flutter run`.
Press **q** to stop. See the parent [README](../README.md) for full details.
