Day 5 — Coding Challenges

Five mini-projects exercising ES6 + DOM manipulation.

  • 1. Live Text Mirror Mirror input text into a paragraph in real time.
  • 2. Color Changer Change the page background with colored buttons.
  • 3. To-Do List Add tasks and delete them via event delegation.
  • 4. Counter Increment, decrement, and reset a number.
  • 5. Form Validator Validate fields on submit with preventDefault.
