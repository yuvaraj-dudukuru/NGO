# Day 5 Coding Challenges — DOM Mini-Projects

Five focused DOM mini-projects, each in its own folder, plus a landing page that
links to all of them.

## Files & folders

```
day5-challenges/
├── index.html              ← landing page linking to all five challenges
├── styles.css              ← shared styling for the landing page
├── 1-live-text-mirror/     ← echo input text live
├── 2-color-changer/        ← change the page background color
├── 3-todo-list/            ← add / delete tasks
├── 4-counter/              ← increment / decrement / reset a number
└── 5-form-validator/       ← validate a sign-up form
```

| # | Challenge | Key concept |
|---|-----------|-------------|
| 1 | **Live Text Mirror** | `input` event → real-time text update |
| 2 | **Color Changer** | setting styles via the `style` property |
| 3 | **To-Do List** | `createElement` + **event delegation** to delete |
| 4 | **Counter** | `let` state + `+` / `-` / `Reset` listeners |
| 5 | **Form Validator** | `preventDefault()` + validation messages |

## How to run

1. Open `index.html` (this folder) to get the landing page, then click into any
   challenge — **or** open a challenge's own `index.html` directly.
2. Any modern browser works (double-click or use **Live Server**).

## How to stop

Close the browser tab (or stop Live Server from the VS Code status bar).

Each sub-folder has its own README with full details.
