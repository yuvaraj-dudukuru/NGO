# ShopSmart — Weekly Sales Summary

A small Python program that takes one week of daily sales figures and produces a quick sales summary. Built as a mini data-analyst challenge.

## Problem

Given the daily sales (in rupees) for Monday to Sunday:

```
3200, 4500, 2800, 5100, 6200, 7400, 5600
```

The program calculates and prints:

1. Total sales for the week
2. Average daily sales
3. Highest and lowest sales day amounts
4. Difference between the highest and lowest day

## Expected Output

```
Total weekly sales: 34800
Average daily sales: 4971.428571428572
Highest sales day: 7400
Lowest sales day: 2800
Difference between best and worst day: 4600
```

## Stretch Goals (also included)

- Round the average to two decimals using `round(average, 2)`
- Identify which weekday had the highest / lowest sales
- Count how many days had above-average sales

## How to Run

**Python script:**

```bash
python sales_summary.py
```

**Jupyter Notebook:**

```bash
jupyter notebook sales_summary.ipynb
```

Then run the cells top to bottom.
