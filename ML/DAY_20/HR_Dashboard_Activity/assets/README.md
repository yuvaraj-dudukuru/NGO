# `assets/` — HR dashboard data bundle

| File | What it is |
|------|-----------|
| `data.js` | The employee dataset as `window.HR_DATA = [ ... ]`, loaded by `../dashboard.html` / `../index.html` so the dashboard runs offline |

`data.js` is **generated** (do not edit by hand) by
[`../data/generate_data.js`](../data/generate_data.js) — see
[`../data/README.md`](../data/README.md). Synthetic employees only — no real people.
